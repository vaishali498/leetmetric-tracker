# LeetMetric Tracker

[Live Demo]

A clean, responsive, and interactive **LeetCode Stats Tracker** built using **HTML, CSS, and JavaScript**.  
Enter any LeetCode username to view **Easy / Medium / Hard problem stats** along with dynamic progress bars, inspired by the LeetCode theme (Minimal White + Orange).

---
## Features

- Enter any **LeetCode username** to view the profile.  
- Dynamic **Easy / Medium / Hard stats** displayed as **progress bars + solved/total/percentage**.  
- Minimal White + Orange **professional UI** with hover animations.  
- Fully **responsive design** for desktop and mobile.  
- Uses **Fetch API** to get real-time stats from the public LeetCode API.  
- Dynamic profile redirection to the entered username.

---

## Tech Stack

- **HTML5**  
- **CSS3** (Responsive, Flexbox)  
- **JavaScript** (Vanilla JS, Async/Await, DOM Manipulation)  
- **Public API**: [LeetCode Stats API](https://leetcode-stats-api.herokuapp.com/)
